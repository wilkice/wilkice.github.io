我使用的那些软件和服务

发表于 2019-06-02 更新于 2019-08-11

桌面软件：

  • Gridea: Markdown 静态博客写作客户端，可搭配 Github Pages
  • Anki: 搭配记忆曲线的 Flash Cards
  • Picgo: 图床客户端，可一键上传图片并复制链接到剪贴板
  • Typora: 颜值超高的 Markdown 写作软件
  • Vs code: 轻量编辑器，支持多种语言
  • Inoreader: RSS 阅读器
  • Pocket: 稍后读应用，可搭配 IFTTT
  • IDM: Windows 下下载工具
  • MobaXterm: Windows 下好用的 SSH 客户端，支持 WSL
  • Fira Code 字体：程序员专用字体，哈哈
  • Telegram
  • Kindle
  • V2rayNG, ss

手机软件：

  • IFTTT
  • Inoreader
  • Good lock( Samsung only)
  • 闲鱼
  • Microsoft to-do
  • i深圳
  • Gboard
  • 知识星球
  • Money Manager：很好用的安卓记账软件，地址是 Google Play Store 的

命令行工具：

  • Youtube-dl

网站：

  • vast.ai: GPU共享平台
  • PDF DRIVE: 原版英文电子书下载网站，可推送到 Kindle
  • Unsplash: 免费、无版权的高清图片资源网站
  • Cloudflare: DNS 域名解析及 CDN 加速，HTTPS 证书加密
  • Medium

订阅服务：

  • Onedrive 家庭版：年付50元左右
  • Spotify 家庭版
  • Datacamp
  • CCF 和 ACM 会员：400/年，主要是 ACM 会员可以免费使用 Safari

网络服务：

  • MongoDB Atlas

    免费的 MongoDB 云存储，存储容量 500 MB

  • GCP SQL Server

    最低配置，用的时候再打开

工具

  • DJI GO

  • Inoreader
