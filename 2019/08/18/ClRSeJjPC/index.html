从0到1安装最纯净的 Windows 系统

发表于 2019-08-18

如何傻瓜式的安装最纯净的 Windows 系统且全程只使用一个官方工具，无 x60全家桶，腾x全家桶😂

准备工作：

  • 一台能上网的 Windows 电脑
  • 一个容量 >= 8G 的 U 盘（此U盘资料会被清空，请提前备份）PS：如果你的电脑有 USB 3.0 接口，请把 U 盘插在该接口上，该 USB 接口里面为蓝色

安装过程如下：

  1. 在微软官网下载介质创建工具，此工具亦可用于本地计算机系统的升级

  2. 打开刚刚下载的软件

  1. 选择第二个：创建媒体介质

  1. 选择 Windows 版本

    ==注意这里要先取消勾选下面的使用本机的推荐设置，然后选择语言，版本（就选第一个 Windows 即可，这个可以安装专业版，家庭版等多个系统），系统架构（选择64位，现在电脑基本都是 64 了）==

  2. 选择安装到U盘

  1. 选择 U 盘

  1. 下载系统中

  1. 创建启动 U 盘中

  1. U 盘创建成功，弹出 U 盘即可，接下来进入到系统安装环节

安装系统环节

此处涉及到的设置较多，请在时间充裕的情况下阅读🤣

进入 BIOS 设置。

我这里以微星主板为例，其余主板基本操作是一样的。微信主板进入 BIOS 是在电脑启动屏幕出现画面之后按下 Del或是Delete，按一下就行了，不行的话就快速的多按几下，不要长按，不要长按，不要长按

笔记本请在网上百度你的笔记本品牌+型号如何进入 BIOS，台式机百度你的主板如何进入 BIOS，


先了解下 BIOS 菜单的基本操作方式

  • Enter：选择及确认
  • ->及<-：切换最上面的主菜单
  • Esc：返回
  • ：保存截图到 U 盘

    先设置下语言

    选中光标所指的地方，按下Enter，选择中文即可

    设置 XMP，即内存超频（可选且不是所有主板都有）

    关于什么是 XMP 网上已有很多专业解释，不在此赘述。
    首先按下右方向键选择菜单到我这里显示的Overclocking。如果你发现这个页面显示的已调内存频率和你买的内存包装上写的频率不一样，比如你买的内存是 3000 MHZ 的，但是已调内存频率不是显示的3000而是2400或是其他数字，你需要手动选择超频文件。在 A-XMP或是其他包含 XMP的选项处选择file 1 或者 file 2，选完后看下**已调内存频率是否有变化。我这里因为之前已经设置了 XMP 所以进来就显示的3000，如果没设置的话应该是2400.

    临时选择 U 盘启动

    在这里选择你的 U 盘并保存重新启动

    系统安装

    系统安装过程这里就不展开了，主要是不好截图🤷‍♂️
