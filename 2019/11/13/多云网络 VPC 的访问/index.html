免密输入

堡垒机实现方法

  1. 新增网域添加界面（接入主机所属的区域，如阿里云上海，华为云深圳），后台需添加数据库表（网域对应接入主机）

    各个网域内可以添加「接入主机」，利用接入主机来访问内部服务器

  2. 新增接入主机添加页面

  3. 新增资产详情添加网域选项

当用户如小王通过堡垒机管理界面连接某一公有云或私有云的内网主机例如 192.168.1.2 时，首先通过数据库查询该资产对应的网域，从其网域下选择一个可用的接入主机如 183.61.34.2，利用该主机作为跳板连接至目标服务器 192.168.1.2，具体命令见上方示例。

堡垒机

ssh -L 3389:内部机:3389 root@接入机

ssh - local and 接入机 访问local = 访问内部机，内部机可以==接入机

接入机

ssh -R 3389:内部服务器:3389 root@提供windows远程服务的主机
